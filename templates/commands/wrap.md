# /wrap — 会话收尾

**第一步：自动写入（无需确认，始终执行）**
向 memory-bank/session-notes.md 末尾追加：
```
---
时间: [当前时间]
任务: [一句话]
完成: [bullet list]
未完成: [bullet list，如有]
---
```

**第二步：生成草稿（输出给我审核，不自动写入）**
a) activeContext.md 更新草稿：
```
[新的"刚完成"和"下一步"内容]
```

b) 如果本次会话有值得记录的教训，输出 lessons.md 新增条目草稿：
```
- [日期] [问题描述] → [解决方案]
```

**第三步：等待我的指令**
- 说"更新上下文"→ 写入 activeContext.md
- 说"记录教训"→ 追加到 lessons.md
- 说"两个都要"→ 两个都写入
- 说"跳过"或直接关闭 → 只保留 session-notes.md 的自动记录

这样设计：即使你最累的时候直接关闭会话，session-notes.md 的记录也已经完成。
